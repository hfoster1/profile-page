# profile-page

Personal profile site for Harry F. Foster — hosted via GitHub Pages.

## Live site

Once Pages is enabled on this repo, the site will be served at:
**https://hfoster1.github.io/profile-page/**

## Enabling GitHub Pages

1. Push this repo to `main`.
2. In GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to `Deploy from a branch`.
4. Select branch `main`, folder `/ (root)`, and save.

## Structure

- `index.html` — single-file site (HTML + embedded CSS + minimal JS).
- No build step, no dependencies.

## Editing

Open `index.html` in any editor. Sections are clearly marked: hero, about, experience timeline, skills, education, interests, footer.
